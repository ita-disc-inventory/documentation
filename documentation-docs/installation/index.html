Installation

This guide will walk you through the process of setting up the ITA application on your local development environment.

Prerequisites​

Before installing the ITA Inventory web app, ensure you have the following installed:

Tool(Minimum) VersionCheck command
Node.js18.0node -v
npmLatest LTSnpm -v
GitAny non-deprecated versiongit --version
  • Node.js (v16 or higher)
  • npm (v8 or higher) or Yarn
  • Git for cloning the repository

Clone the Repositories​

Clone each repository separately:

Frontend

git clone https://github.com/ita-disc-inventory/frontend.git

Backend

git clone https://github.com/ita-disc-inventory/backend.git

Backend Setup​

Navigate to the backend directory and install dependencies:

cd backend
npm install

Frontend Setup​

Navigate to the frontend directory and install dependencies:

cd ../frontend
npm install

Running the Application​

Start the Backend Server​

cd backend
npm run dev

The backend server will start on http://localhost:3000.

Start the Frontend Development Server​

In a new terminal:

cd frontend
npm start

The frontend development server will start on http://localhost:3001. You should be able to navigate to this URL and view the ITA Inventory application.

Troubleshooting​

If you encounter issues during installation:

  1. Ensure all prerequisites are correctly installed
  2. Check that all dependencies were properly installed
  3. Ensure required ports (3000, 3001) are available
